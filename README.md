# JustHazzardous - Gaming Website

A dynamic gaming website featuring game reviews, news, and featured titles. Built with pure HTML and CSS for a sleek, responsive design.

## Features

- **Hero Section**: Eye-catching introduction with gaming background
- **Featured Games**: Showcase of popular and upcoming games
- **Reviews Section**: Latest game reviews and ratings
- **Gaming News**: Current news and rumors from the gaming world
- **Responsive Design**: Optimized for desktop and mobile devices

## How to View

Open `index.html` in your web browser to view the website.

## Optional Local Server

Run a local server from the project folder:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Technologies Used

- HTML5
- CSS3 (with modern features like CSS Grid and Flexbox)
- Responsive design principles
